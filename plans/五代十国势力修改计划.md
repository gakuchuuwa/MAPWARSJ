| 分类 | 操作 | 详情 |

| ✅ 新建 `dongdan` | 东丹国 | 据点：扶余城（接管现有 `city_fuyu`） |

## 步骤 1：修改 factions.ts

### 在现有 faction 列表中添加3个新势力

推荐位置：在 `qian_d`、`lyu_d`、`yuwenhuaji` 之后（或按字母顺序插入合适位置）。

```typescript
// 新势力添加示例（具体位置由实施者确定，推荐在 xijue/xian_d 之后，xueyantuo 之前）
{ id: 'dongdan', name: '东丹', color: '#1E5B8A' },      // 渤海蓝 - 东丹国(耶律倍)
{ id: 'dali', name: '大理', color: '#7B3F9E' },          // 紫霞紫 - 大理国(段氏)
{ id: 'luodian', name: '罗甸', color: '#8B5A2B' },       // 土褐 - 罗甸国(西南蛮)
```

### 颜色说明
- `dongdan` — 深蓝 (#1E5B8A)，渤海/东丹为北方海裔
- `dali` — 紫 (#7B3F9E)，大理段氏佛教王国特色
- `luodian` — 土褐 (#8B5A2B)，西南山地部落风格

## 注意事项

1. **扶余城冲突**：`fuyu` 夫余国的唯一城市被东丹接管后，`fuyu` faction 将失去所有城市。如后续需要还原，需新增其他城市给 `fuyu`。
2. **大理距蒙舍城约53km**：略超50km规则，技术上可行。如有边界问题可微调坐标。
3. **回鹘据点改为斡耳朵八里**：甘州（张掖）已被 `juqu_d`（北凉）占据；西州（高昌）已被 `gaochang` 占据。改用回鹘汗国漠北首都斡耳朵八里（47.30, 102.65），距薛延陀郁督军山约54km ✅ >50km。
4. **吐谷浑无需修改**：已有伏俟城（`city_fusicheng`）作为首都，与用户提供的坐标一致。
