# 据点间距优化建议

> 基于 [`src/data/cities_v2.ts`](src/data/cities_v2.ts) 的数据分析
>
> 优化原则：**以历史名城为主** — 当两个据点间距 < 50km 时，保留历史上更著名、更核心的城市，删除或合并次要的据点。

---

## 总览

| 指标 | 数值 |
|------|------|
| 总城市数 | 556 |
| 违规对数 (间距 < 50km) | 178 |
| 建议删除数 | ~40-50 |
| 完全重复/同城异名 | ~25 对（距离 < 5km） |
| 历史真实近距据点 | ~50 对（保留） |

> **核心原则**：以上建议以**保留历史名城**为最高标准。当某对据点是同一城市的古今不同名称时，保留更古老的名称；当某对据点是历史上真实相邻的不同城市时，即使间距 < 50km 也予以保留。
