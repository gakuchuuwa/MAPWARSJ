# 巨鹿之战纪实

## 序幕：秦末乱世

公元前208年，秦帝国已入暮年。陈胜、吴广揭竿而起，天下响应，群雄并起。然而，秦将章邯率军平定关中叛乱后，挥师东进，连破周文、田儋、项梁等诸侯军，声威大震。秦军锋芒所向，六国遗民无不胆寒。

此时，赵国成为秦军东进的首要目标。章邯麾下大将王离、涉间率二十万秦军，围困赵王歇于巨鹿城中。城外，秦将章邯亲率主力，筑甬道以通粮道，形成内外夹击之势。巨鹿城岌岌可危，赵国危在旦夕。

## 第一章：诸侯观望

赵将陈余率数万兵马驻扎巨鹿北方，却不敢与秦军正面交锋，只能眼睁睁看着巨鹿被围。赵相张耳困守城中，屡次遣使向陈余求援，言辞恳切："将军若能率军入城共守，尚有一线生机；若不能入城，请速发兵攻击秦军，以解巨鹿之围。"

然而，陈余深知秦军之强，不敢轻举妄动。他对使者叹道："吾所部不过数万，秦军却有数十万之众。若贸然出击，无异于以卵击石，不仅救不了巨鹿，反而会全军覆没。"

赵国的危急传遍天下。楚怀王心王遣宋义为上将军，项羽为次将，范增为末将，率楚军北上救赵。然而，宋义行至安阳，便停军不前，一驻就是四十六日。他的理由冠冕堂皇："秦军正盛，不可轻敌。不如待秦赵相斗，两败俱伤，我军再坐收渔利。"

项羽却不这样想。这位年轻的将领目睹秦军暴虐，心中怒火难平。他多次向宋义进谏，请求速战，却屡遭拒绝。宋义甚至下令："军中有猛如虎、狠如羊、贪如狼、倔强不听指挥者，一律斩首！"这道命令，分明是针对项羽而发。

## 第二章：破釜沉舟

公元前207年冬，天寒地冻。项羽再也无法忍受宋义的拖延。一个清晨，他径直闯入宋义帐中，拔剑斩杀了这位上将军。随后，他召集诸将，宣布："宋义与齐国勾结，图谋不轨，楚王密令我诛杀叛徒！"

诸将慑于项羽的威势，无人敢反对。楚怀王心王得知此事，虽然震怒，却也无可奈何，只得追认项羽为上将军，统领楚军。

项羽接掌兵权后，立即展现出截然不同的作风。他先遣英布、蒲将军率两万精锐渡河，切断秦军甬道，初战告捷。随后，项羽亲率主力渡过漳河。

渡河之后，项羽做出了一个惊人的决定：他下令士卒凿沉所有渡船，打破所有炊具，烧毁所有营帐，每人只携带三日口粮。这就是历史上著名的"破釜沉舟"。

将士们明白了将军的决心——此战只能进，不能退；只能胜，不能败。没有退路，唯有死战！

## 第三章：九战九捷

楚军以必死之心，向秦军发起猛攻。项羽身先士卒，冲锋在前。史书记载："楚战士无不一以当十，声震天地。"

秦军虽然人数众多，却未料到楚军如此悍勇。王离军首当其冲，在楚军的猛烈攻势下节节败退。项羽率军连战连捷，一日之内九战九胜，彻底击溃了围困巨鹿的秦军主力。

此时，诸侯联军虽然云集巨鹿周边，却都筑营自保，无人敢出战。他们站在营垒上，目睹楚军以少胜多的壮举，无不震惊。

战后，项羽召见诸侯将领。史书记载："诸侯将入辕门，无不膝行而前，莫敢仰视。"从此，项羽实际上成为了诸侯联军的统帅，号令天下。

## 第四章：秦军覆灭

巨鹿之战后，秦军主力遭受重创。王离被俘，涉间自焚而死，苏角战死。章邯虽然保存了一部分兵力，却已无力回天。

项羽乘胜追击，连续攻克秦军营垒。章邯节节败退，从巨鹿退至棘原，又从棘原退至漳水南岸。秦军士气低落，战斗力大减。

更糟糕的是，咸阳朝廷对章邯失去了信任。赵高当权，不仅不发援军，反而派人责备章邯。章邯深知，即使战胜归来，也难逃一死。他的副将司马欣曾回咸阳汇报军情，却连赵高的面都见不到，在宫门外等了三天三夜。司马欣惊恐而归，对章邯说："赵高专权，我等无论胜败，都是死路一条。"

走投无路的章邯，最终选择了投降。公元前207年七月，章邯率二十余万秦军向项羽投降。这支曾经横扫六国的秦军主力，就此覆灭。

## 尾声：历史转折

巨鹿之战，是秦末战争的转折点。此战之后，秦军主力尽失，再也无力阻挡诸侯联军的进攻。项羽一战成名，成为反秦联军的实际领袖。

这场战役充分展现了项羽的军事才能：他果断诛杀宋义，掌握兵权；破釜沉舟，激发士气；身先士卒，以少胜多。然而，这场胜利也埋下了日后的隐患——项羽坑杀二十万秦军降卒，种下了与秦地人民的深仇大恨。

巨鹿之战后不到一年，刘邦攻入咸阳，秦帝国灭亡。而项羽与刘邦之间的楚汉之争，也就此拉开序幕。

**史家评论**

《史记》评价项羽："项羽非有尺寸之地，起于行伍之中，三年遂将五诸侯灭秦，分裂天下而封王侯，政由羽出，号为霸王。位虽不终，近古以来未尝有也。"

巨鹿之战，正是项羽军事生涯的巅峰。这场以少胜多的经典战役，与淝水之战、官渡之战并称中国古代三大以少胜多的战役，永载史册。
