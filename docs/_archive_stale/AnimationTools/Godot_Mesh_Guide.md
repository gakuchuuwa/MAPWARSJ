# Godot“网格绑定”操作指南 (最后一步)

你现在的界面里应该是一片黑或者空的，因为我们需要手动把图片“画”出来。

请按顺序操作：

## 1. 让人物显示出来 (绘制网格)
1.  在左侧场景面板，选中 **`BodyMesh`** 这个节点。
2.  在上方工具栏，找到一排小图标，点击那个 **✏️ (笔形状，或者叫 Points)** 图标。
3.  **开始画轮廓**：
    *   在主视图里，沿着你的士兵身体边缘点击鼠标左键。
    *   把头、手、脚、身体大概围一圈。
    *   最后点回起点闭合。
    *   **瞬间**！你的士兵图片应该就会显示出来了。

## 2. 对齐骨头 (Align Bones)
1.  图片显示出来后，你会发现我预设的骨头（灰色的线）可能跟你的士兵位置不重合。
2.  选中 **`Skeleton2D`** 下面的骨头节点（比如 `Head`, `Arm_L` 等）。
3.  使用 **移动工具 (W)** 和 **旋转工具 (E)**，把骨头拖到它该在的位置（比如把 `Head` 骨头拖到士兵头上）。

## 3. 绑定 (灵魂注入)
1.  再次选中 **`BodyMesh`**。
2.  在上方工具栏点击 **UV** 按钮打开通过面板。
3.  点击菜单里的 **Bones (骨骼)** -> **Sync Bones to Polygon (同步骨骼)**。
4.  此时网格会变色，代表骨头已经控制了身体！

---
做完这三步，试着随便转动一根骨头，如果你的士兵身体跟着动了，我们就成功了！
