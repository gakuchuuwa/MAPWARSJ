## 🎯 测试步骤

### 第一步：打开游戏

1. 打开浏览器（推荐 Chrome 或 Edge）
2. 访问：http://localhost:3000
3. 按 F12 打开开发者工具

### 第二步：检查控制台

在控制台（Console）中，您应该看到：
```
🎮 MAPWAR 游戏已启动
📊 使用 Vite 开发服务器
🔥 热重载已启用
```

如果看到这些信息，说明基础环境正常。

### 第三步：检查游戏画面

**应该看到：**
- ✅ 黑色背景的游戏画面
- ✅ 左上角的 HUD 面板（显示城市、部队数量）
- ✅ 控制按钮（开始、暂停、重置、速度）
- ✅ 地图图片（中国地形图）
- ✅ 城市标记（彩色圆点）

**如果地图没有显示：**
- 检查控制台是否有图片加载错误
- 这是正常的，因为图片可能需要几秒钟加载

### 第四步：测试基础功能

1. **点击"开始"按钮**
   - 游戏应该开始运行
   - 时间应该开始流动
   - 季节进度条应该移动

2. **测试地图交互**
   - 拖拽地图：按住鼠标左键拖动
   - 缩放地图：使用鼠标滚轮

3. **观察历史事件**
   - 等待一段时间
   - 应该会弹出历史事件通知
   - 地图上会出现军队移动

### 第五步：测试热重载

1. 保持浏览器打开
2. 在 VS Code 中打开 `style.css`
3. 修改一个颜色，例如：
   ```css
   .hud-title {
       color: #ff0000; /* 改成红色 */
   }
   ```
4. 保存文件（Ctrl+S）
5. 观察浏览器是否自动刷新
6. HUD 标题应该变成红色

## 🐛 常见问题排查

### 问题1：页面空白
**可能原因：**
- JavaScript 错误
- 资源加载失败

**解决方法：**
1. 打开控制台查看错误信息
2. 检查 Network 标签，查看哪些资源加载失败
3. 截图发给我

### 问题2：地图不显示
**可能原因：**
- 图片路径错误
- 图片加载慢

**解决方法：**
1. 等待 5-10 秒
2. 检查控制台是否有图片加载错误
3. 查看 Network 标签，找到图片请求

### 问题3：控制台有错误
**解决方法：**
1. 复制完整的错误信息
2. 截图发给我
3. 我会立即修复

### 问题4：热重载不工作
**可能原因：**
- 浏览器缓存
- Vite 服务器问题

**解决方法：**
1. 刷新浏览器（Ctrl+F5 强制刷新）
2. 重启 Vite 服务器：
   - 在终端按 Ctrl+C 停止
   - 运行 `npm run dev` 重启
