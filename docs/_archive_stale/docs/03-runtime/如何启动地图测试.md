# 如何启动地图测试

## ❌ 问题：CORS错误

直接双击HTML文件会遇到CORS错误，因为浏览器不允许本地文件使用ES6模块。

## ✅ 解决方法：使用本地服务器

### 方法1：一键启动（最简单）

**双击运行这个文件：**
```
C:\Users\GAKU\Desktop\MAPWAR\start_server.bat
```

会自动：
1. 启动Python本地服务器
2. 在8000端口运行
3. 显示访问地址

然后在浏览器打开：
```
http://localhost:8000/test_tilemap.html
```

---

### 方法2：手动启动

1. **打开命令提示符（CMD）**
   - 按 Win+R
   - 输入 `cmd`
   - 按回车

2. **进入MAPWAR目录**
   ```cmd
   cd C:\Users\GAKU\Desktop\MAPWAR
   ```

3. **启动服务器**
   ```cmd
   python -m http.server 8000
   ```

4. **打开浏览器**
   - 访问：`http://localhost:8000/test_tilemap.html`

---

### 方法3：使用其他浏览器

**Firefox浏览器** 允许本地文件使用ES6模块：
1. 下载安装Firefox
2. 直接双击 `test_tilemap.html`
3. 应该可以正常运行

## 🐛 常见问题

### Q1: 浏览器打不开
**A**: 检查服务器是否启动
- 看CMD窗口有没有显示 "Serving HTTP on..."
- 尝试访问：`http://localhost:8000`

### Q2: 地图全是灰色方块
**A**: 瓦片文件路径不正确
1. 按F12打开控制台
2. 查看Network标签
3. 看哪些图片文件404
4. 告诉我错误信息

### Q3: 地图加载很慢
**A**: 正常现象
- 第一次加载会较慢（下载瓦片）
- 之后会使用缓存，速度变快
- 12级瓦片文件很大，加载较慢

### Q4: 控制台有错误
**A**: 请截图或复制错误信息告诉我
