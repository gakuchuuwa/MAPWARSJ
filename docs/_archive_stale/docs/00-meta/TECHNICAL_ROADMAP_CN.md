# MAPWAR 项目技术路线图

## 项目概述
MAPWAR 是一款基于中国历史背景的策略战争游戏，核心玩法包括：
- 历史事件驱动
- 势力发展与扩张
- 战争与外交系统
- 资源管理与城市建设

## 技术架构总览
```
├── 核心引擎: Phaser 3
├── 状态管理: Redux
├── 前端框架: React (可选)
├── 构建工具: Vite
├── 编程语言: TypeScript
├── 数据存储: IndexedDB
└── 网络通信: Socket.IO (多人游戏)
```

## 2. 项目结构

### 2.1 目录结构
```
src/
├── assets/               # 游戏资源
│   ├── images/          # 图片资源
│   ├── sprites/         # 精灵表
│   ├── maps/            # 地图数据
│   └── sounds/          # 音效
├── core/                # 核心游戏逻辑
│   ├── GameManager.ts   # 游戏主循环
│   ├── SceneManager.ts  # 场景管理
│   └── EventSystem.ts   # 事件系统
├── entities/            # 游戏实体
│   ├── City.ts          # 城市类
│   ├── Unit.ts          # 单位类
│   └── Faction.ts       # 势力类
├── scenes/              # 游戏场景
│   ├── BootScene.ts     # 启动场景
│   ├── WorldMapScene.ts # 世界地图
│   └── BattleScene.ts   # 战斗场景
├── systems/             # 游戏系统
│   ├── CombatSystem.ts  # 战斗系统
│   ├── EconomySystem.ts # 经济系统
│   └── AISystem.ts      # AI系统
└── ui/                  # 用户界面
    ├── components/      # UI组件
    └── styles/          # 样式文件
```
