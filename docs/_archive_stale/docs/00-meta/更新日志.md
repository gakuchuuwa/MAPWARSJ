# MAPWAR 更新日志

## 📅 2025年10月31日 19:45

### ✨ 新功能

#### 1. 动态城市创建系统
- ✅ 新增 `spawn_city` 动作类型
- ✅ 支持在游戏进行中动态创建城市
- ✅ 城市可设置为都城、主要城市或次要城市
- ✅ 自动在地图上渲染新城市

**示例：**
```javascript
{
    type: "spawn_city",
    cityId: "hanzhong",
    name: "汉中",
    factionId: "qi",
    latitude: 33.07,
    longitude: 107.02,
    importance: "capital",
    troops: 15000
}
```

#### 2. 势力灭亡系统
- ✅ 新增 `faction_destroyed` 动作类型
- ✅ 记录势力灭亡事件
- ✅ 支持自定义灭亡消息

**示例：**
```javascript
{
    type: "faction_destroyed",
    factionId: "qin",
    message: "秦国灭亡"
}
```

#### 3. 新增历史事件

- 秦国灭亡

## 📅 2025年10月31日 19:30

### 🎨 视觉优化

#### 燕国颜色调整
- ❌ 白色 `#ffffff` → ✅ 浅灰色 `#d3d3d3`
- 原因：白色在地图上不够清晰
- 浅灰色既保持"尚白"文化特征，又确保可见性

#### 浅灰色势力透明度优化
- 边界透明度：0.3 → **0.6** （提高100%）
- 中心透明度：0.1 → **0.4** （提高300%）
- 确保燕国在地图上清晰可见
