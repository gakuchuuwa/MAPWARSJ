# 🗺️ 地形通行层系统 - 使用说明

## ✅ 已完成整合

地形通行层系统已成功整合到MAPWAR主游戏中！

## 🎮 如何使用

### 方法1：快捷键（推荐）
按键盘上的 **T键** 即可快速切换地形通行层显示

### 方法2：界面按钮
点击游戏界面右侧的 **"地形"** 按钮

## 📊 三色分类系统

当开启地形通行层后，地图上会显示三种颜色：

| 颜色 | 地形类型 | 通行速度 | 说明 |
|------|---------|---------|------|
| 🌊 **蓝色** | 海洋 | 0% | 完全不可通行 |
| ⛰️ **棕色** | 山地 | 30% | 慢速通行，行军困难 |
| 🌾 **绿色** | 平原 | 100% | 平速通行，快速移动 |

## ⚡ Zoom 9核心优化

系统针对不同缩放级别自动优化：

- **Zoom 9**：核心层，使用最高精度（4px采样间隔）
- **Zoom 8/10**：高精度（6px采样间隔）
- **Zoom 7/11**：标准精度（8px采样间隔）

在Zoom 9时，打开浏览器控制台(F12)可以看到详细的地形统计信息。

## 🎯 使用场景

1. **战略规划** - 查看地形影响，规划最优行军路线
2. **军事部署** - 识别地形障碍，选择有利防守位置
3. **战场分析** - 评估地形对战斗的影响

## 🚀 快速开始

1. 打开游戏：http://localhost:8000/
2. 按 **T键** 开启地形通行层
3. 滚轮缩放到 **Zoom 9** 查看最高精度效果
4. 打开浏览器控制台(F12)查看统计信息

## 💡 提示

- 地形通行层不影响游戏性能
- 可随时开关，不影响游戏进行
- 支持所有缩放级别(7-11)
- 自动适配不同地形图瓦片
