# 玩家系统实现文档

## ✅ 已完成功能

### 1. 玩家角色系统
- **初始状态**：1人，偏裨将军（9级），功勋0
- **起始位置**：长安（34.34°N, 108.94°E）
- **官职系统**：9个等级，从偏裨将军到大元帅

### 2. 官职等级表
| 等级 | 官职名称 | 最大领兵数 | 所需功勋 |
|------|---------|-----------|---------|
| 9 | 偏裨将军 | 100 | 0 |
| 8 | 威猛将军 | 200 | 100 |
| 7 | 讨荡将军 | 400 | 300 |
| 6 | 振威将军 | 1,000 | 700 |
| 5 | 安平将军 | 2,000 | 1,500 |
| 4 | 征镇将军 | 5,000 | 3,000 |
| 3 | 卫方将军 | 10,000 | 6,000 |
| 2 | 骠骑将军 | 20,000 | 12,000 |
| 1 | 大元帅 | 30,000 | 25,000 |

### 3. 移动系统
- **操作方式**：左键点击地图任意位置
- **移动速度**：0.5度/秒
- **视觉反馈**：
  - 玩家图标：金色五角星 ⭐
  - 移动路径：虚线显示
  - 光晕效果：金色渐变

### 4. UI显示
- **玩家信息面板**（左下角）：
  - 当前官职
  - 兵力 / 最大兵力
  - 功勋值
  - 效忠势力

## 📋 待实现功能

### 1. 势力选择
玩家需要选择效忠的势力才能参战：
```javascript
// 示例：选择效忠汉国
player.joinFaction('qi');
```

### 2. 战斗参与机制
- **攻城战**：玩家靠近敌方城市时，可选择参与攻城
- **野战**：当历史事件触发野战时，玩家可选择参与
- **战斗结算**：
  - 胜利：获得功勋 = 击败敌军数 / 100
  - 失败：无功勋

### 3. 功勋和晋升
- 功勋达到要求自动晋升
- 晋升后最大领兵数增加
- 控制台显示晋升通知

### 4. 招募系统
- 在友方城市招募士兵
- 受官职限制（不能超过最大领兵数）

## 🎮 使用方法

### 当前可用操作
1. **启动游戏**：点击"开始"按钮
2. **移动玩家**：左键点击地图任意位置
3. **查看信息**：左下角查看玩家状态
4. **时间控制**：使用控制按钮或空格键

### 控制台命令（测试用）
```javascript
// 获取玩家信息
game.player.getInfo()

// 增加功勋（测试晋升）
game.player.addMerit(1000)

// 招募士兵
game.player.recruitTroops(50)

// 选择效忠势力
game.player.joinFaction('qi') // 汉国
```

## 🔧 技术实现

### 文件结构
```
src/
├── player/
│   └── Player.js          # 玩家类
├── data/
│   └── playerData.js      # 官职和初始数据
game.js                     # 主游戏类（集成玩家）
index.html                  # UI（玩家信息面板）
style.css                   # 样式（玩家UI）
```

### 核心类：Player
```javascript
class Player {
    position: { latitude, longitude }  // 位置
    troops: number                     // 兵力
    merit: number                      // 功勋
    rank: Rank                         // 官职
    faction: string                    // 效忠势力
    
    moveTo(lat, lng)                   // 移动
    joinFaction(factionId)             // 加入势力
    joinBattle(battle)                 // 参战
    addMerit(amount)                   // 增加功勋
    recruitTroops(amount)              // 招募
}
```

## 🎯 设计理念
- **简单直观**：左键点击即可移动
