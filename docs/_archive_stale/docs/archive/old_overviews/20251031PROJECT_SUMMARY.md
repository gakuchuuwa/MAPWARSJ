## 💡 关键技术决策

### 1. 选择 Vite 而非 Webpack
**理由：**
- 更快的开发服务器启动
- 更好的热重载体验
- 更简单的配置
- 更适合现代浏览器

### 2. 渐进式重构而非完全重写
**理由：**
- 降低风险
- 保持功能完整
- 可随时回滚
- 更快看到成果

### 3. 离屏 Canvas 缓存
**理由：**
- 领土计算耗时（Voronoi算法）
- 城市归属变化不频繁
- 显著提升性能

### 4. 保留 game.js 主体
**理由：**
- 游戏逻辑复杂
- 避免过度拆分
- 保持代码可读性
