## 💡 关键技术决策

### 1. 选择 Vite
**理由：**
- 极快的开发服务器启动
- 优秀的热重载体验
- 简单的配置
- 原生ES模块支持

### 2. 渐进式重构
**理由：**
- 降低风险
- 保持功能完整
- 可随时回滚
- 更快看到成果

### 3. 离屏Canvas缓存
**理由：**
- Voronoi计算耗时
- 城市归属不常变化
- 显著性能提升

### 4. 性能监控工具
**理由：**
- 实时了解性能
- 验证优化效果
- 发现性能瓶颈
