# 历史大乱斗 — 武将技（General Skills）系统设计与维护约定

> **必读**：本文档旨在指导未来的 AI 或维护者如何为《历史大乱斗》（MAPWARSJ）添加和维护武将技系统。

## 一、系统核心设计原则（2026-06 确立）

为解决剧本军团长途远征战损过大的问题，并展现名将风采，确立以下设计底线与原则：
1. **统一标准乘区（`*1.2` 铁律）**：所有涉及常规战力加成的技能，底层一律使用 `*1.2` 为标准乘区，允许多个条件叠加。
2. **将领只随军出征**：武将技严格绑定在出征的军团上。将领不驻扎守城，因此没有“守城”加成。
3. **系统拆分**：每个将领的技能体系由 **【战术层】（交锋瞬间的爆发）** 与 **【战略层】（环境与大地图的维系）** 两个相互独立的模块组成。

---

战术技能 = **`时机` × `效果`**，共 **10 种**。

## 三、战略层 · 七格（国民认知版）

战略技能不需要在战场中弹华丽 UI，但决定了军团在大地图的生存与环境适应力。由于将领只带军出征不守城，战略层精准划分为 7个核心技能：

| 类型 | 效果 | 技能名 |
|:---|:---|:---|
| 军机运营 | 速度×1.2 | **兵贵神速** |
| 军机运营 | 兵力+20% | **因粮于敌** |
| 城野专精 | 攻城×1.2 | **攻城拔寨** |
| 城野专精 | 野战×1.2 | **所向披靡** |
| 地形专精 | 平原×1.2 | **长驱直入** |
| 地形专精 | 山地×1.2 | **居高临下** |
| 地形专精 | 水域×1.2 | **乘风破浪** |

## 四、技能装配示例与开发纪律

通过 **【1个战术技】 + 【1个战略技】** 的组合，即可完美定义一个极具历史特色的武将。

**【示例装配】**
*   **白起（秦之锐士）**：③ **【侵掠如火】**（开局自身战力 `*1.2`） + **【因粮于敌】**（战后恢复兵力）。战无不胜，越打越强，平推长平与邯郸。
*   **王翦（大秦铁甲）**：⑤ **【不动如山】**（开局己方免伤） + **【攻城拔寨】**（攻城战力 `*1.2`）。绝对的攻坚机器与铁壁阵型。
*   **邓艾（魏武卒）**：⑦ **【攻其不备】**（逆局扣除敌方兵力） + **【居高临下】**（山地战力 `*1.2`）。绝境翻盘的山地突袭之神。

**开发纪律要求：**
1. **最小侵入原则**：所有武将技必须严格复用上述框架，作为现有系统的增量插件存在。严禁为了某一个神将大改基础战斗管线。
2. **严守 `*1.2` 红线**：所有持续性战力乘区只认 `*1.2`，确保体系数值平衡不崩盘。
