## ⚙️ 一势力一城模型

### 核心规则

每个势力对应 **一个城市**（`one-city-per-faction`）。数据层表现为：
- 每个 `factionId` 在 `cities_v2.ts` / `cities.ts` 中出现且仅出现一次
- 势力被消灭 = 其唯一城市被占领
- 势力无城可退 = 彻底出局

### 数据字段

```
势力 (Faction):
  id: string       — 唯一标识符（如 'qin', 'xiongnu', 'wang_d'）
  name: string     — 显示名称
  color: string    — 十六进制颜色代码

城市 (CityDataV2):
  id: string       — 城市唯一ID
  name: string     — 城市名称
  factionId: string — 所属势力ID（一个势力一个城）
  lat/lng: number  — 真实地理坐标
  type: CityType   — 大城/中城/小城/关隘/渡口/战场
  tier: 0|1|2|4    — 层级
```

## 🎨 颜色对比

颜色选择遵循以下原则：

1. **高对比度**：相邻势力使用对比鲜明的颜色
2. **历史感**：颜色符合传统认知（秦黑、明红等）
3. **分类识别**：可通过颜色大致判断势力类型

### 颜色分组

**深色系（政权-厚重感）：**
- 秦国：黑色 `#000000`
- 明国：红色 `#ff0000`
- 宋国：蓝色 `#4169e1`
- 吐蕃：土黄 `#cd853f`

**亮色系（民族-自然色）：**
- 契丹：天蓝 `#1e90ff`
- 突厥：暗红 `#8b0000`
- 匈奴：金黄 `#ffd700`

**柔和色系（大家族-文雅色）：**
- 琅琊王氏：淡紫 `#9370db`
- 兰陵萧氏：紫罗兰 `#ba55d3`
- 清河崔氏：靛蓝 `#7b68ee`
